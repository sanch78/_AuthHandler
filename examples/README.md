# AuthHandler Examples

This directory contains generic example integration files.

These are not drop-in production files. They are minimal reference examples intended to show structure, responsibilities, and naming.

Current example set:

- `basic-php-app/`: a minimal PHP application layout that wires AuthHandler into a page request lifecycle

Use these examples together with:

- `docs/integration-guide.md`
- `docs/module-usage-reference.md`
